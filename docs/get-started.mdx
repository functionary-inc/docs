```json
{
  "models": [
    {
      "name": "organization",
      "trigger_on_first_identify": {
        "notifs": { "destination": "slack", "channel": "sales-funnel" },
        "webhooks": [{ "method": "GET", "url": "http://anockableURL.com" }]
      }
    },
    { "name": "customer" }
  ],
  "actions": [
    {
      "id": "onboarding_funnel",
      "on_chronological_events": [
        { "event": "signed_up" },
        { "event": "completed_profile" },
        {
          "event": "started_first_stream",
          "trigger": {
            "notifs": [{ "destination": "slack", "channel": "sales-funnel" }]
          }
        },
        {
          "event": "had_first_attendee",
          "trigger": {
            "notifs": { "destination": "slack", "channel": "sales-funnel" },
            "webhooks": { "method": "POST", "url": "http://anockableURL.com" }
          }
        }
      ]
    }
  ]
}
```

Use events from anywhere (crm, code, zaps, airtable, etc.) to track your leads from sign-up through integration. Built for b2b startups selling to customer through slack connects.
